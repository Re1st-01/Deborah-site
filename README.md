# Site Deborah — Consultora Imobiliária (React + Vite)

Projeto inicial scaffold para o site da Deborah — Consultora Imobiliária, usando React + Vite.

Como rodar localmente:

1. Instalar dependências

```powershell
npm install
```

2. Rodar em desenvolvimento

```powershell
npm run dev
```

3. Build de produção

```powershell
npm run build
npm run preview
```
